## Overview

Simplicity-focused design with ample whitespace, legible typography, and a limited color palette to reduce visual clutter. Uses Klima typeface for a distinctive geometric sans-serif feel paired with the warmth of amber/brown brand colors.

## Style Foundations

- **Visual style:** minimal, clean, Swiss editorial
- **Typography scale:** 11/13/14/16/20/24/28/36/40/48/56
- **Typography font:** Klima (Light 300, Regular 400, Medium 500, Bold 700)
- **Color palette:** amber (primary), brown (secondary), cream (surface alt), near-black (text)
- **Spacing scale:** 8pt baseline grid
- **Border radius:** 4px (sm), 8px (md) — no large radii
- **Shadows:** minimal, used sparingly for elevation

## Colors

- **Amber (#FFB23E):** Primary brand color — CTAs, accents, dividers, highlights
- **Brown (#534B38):** Secondary brand — text emphasis, UI elements
- **Cream (#FFF7EB):** Warm surface alternative for section backgrounds
- **Near-black (#1f1f1f):** Primary text color, high contrast
- **Amber-dark (#D58816):** Hover/active states for amber elements
- **White (#FFFFFF):** Default surface
- **Gray-50 (#F9FAFB):** Alternating section background

## Components

- **Buttons:** Sentence case, rounded-md (8px), font-weight 500
- **Inputs:** rounded-md (8px), 1px border, subtle focus ring
- **Cards:** rounded-md (8px), 1px border border-black/5, minimal shadow
- **Links:** Underline with offset, amber decoration
- **Section labels:** Uppercase, letter-spacing 3px, font-weight 500, small size
- **Dividers:** 40px wide amber line, 2px height
